# Shelter
It's the game named "Shelter"

## How to use?

### First way
You can just download `ShelterGame.exe` and start it on your device.

### Second way
You can download `main.py` and `shelter.py`.  
You should put it in one directory and start `main.py` by Python  
```
python main.py
```
